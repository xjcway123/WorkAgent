# 模板解剖（template.pptx）

南通大学风格毕业答辩模板的完整结构、设计参数与内容页档案。规划幻灯片映射、编辑每页时参考本文件。

## 整体规格

- 尺寸：16:9（`cx=12192000 cy=6858000` EMU，13.333" × 7.5"）
- 原件共 25 页，对应一篇硕士答辩（题目《基于大语言模型的软件漏洞分析方法的研究》）。这些页同时是**设计源**和**内容档案样例**。

## 两套版式（slideLayout）—— 设计灵魂所在

视觉装饰全部来自版式，新建/复制页只要挂对版式就自动继承装饰。

| 版式 | 角色 | 装饰来源 | 用于 |
|------|------|----------|------|
| **slideLayout4** | branded（品牌页） | `image4.jpeg`=蓝色 3D 立方体满版背景（聚集在左上）；`image5.png`=南通大学校徽（右上） | 封面、目录、所有章节分隔页、结束页 |
| **slideLayout1** | content（内容页） | `image2.png`=左上角小圆形校徽；`image3.png`=底部页脚条 | 几乎所有内容页 |
| **slideLayout2** | content 变体 | 同上 | 致谢内容页 |

复制 branded 页时，立方体与校徽随版式而来，无需手动加图。

## 配色与字体（设计令牌）

主题色（`ppt/theme/theme1.xml`）：
- `dk2` / 深蓝 = **`293764`** —— 章节分隔大数字、深色标题文字
- `accent1` = `lt2` = 品牌蓝 = **`4A70AE`** —— 内容标题、强调、链接
- 背景 = 白 `FFFFFF` + 立方体装饰
- 强调红 `FF0000` 偶用于关键词高亮（克制使用）

分隔页文字用 `schemeClr val="bg2"`（映射到深蓝）。

字体：
- 中文：**微软雅黑**（`微软雅黑`，`charset="-122"`）
- 数字/英文：**Arial**
- 标题加粗 `b="1"`

字号（半磅制，XML 里写 sz 值=磅数×100）：
- 章节分隔大数字“01”：`sz="12800"`（128pt）
- 章节分隔章节名：`sz="4300"`（43pt）
- 内容页标题：约 28–32pt，蓝色加粗
- 正文/要点：14–18pt
- 参考文献脚注：约 9–11pt 灰色小字

## 关键页 XML 改法速查

### 章节分隔页（如 slide3）
两个文本框：
- 大数字框：`<a:t>01</a:t>`，`sz="12800" b="1"`，字体 Arial，色 `schemeClr val="bg2"`。同时改紧随的 `<a:endParaRPr>`（虽不显示，保持一致）。
- 章节名框：`<a:t>研究背景</a:t>`，`sz="4300" b="1"`，微软雅黑，色 `bg2`。
复制此页做新分隔页时，只改这两处文本。

### 封面页（slide1）
含题目、答辩人/指导老师、学院/实验室三组文本框。逐框 str_replace 替换文本，保留字号与位置。

### 内容页标题
内容页标题多为占位符继承样式（蓝色加粗）。改 `<a:t>` 文本即可；若需强制蓝色加粗，在 `<a:rPr>` 加 `b="1"` 与 `<a:solidFill><a:srgbClr val="4A70AE"/></a:solidFill>`。

### 多条要点
每条一个 `<a:p>`，从原段落复制 `<a:pPr>` 保持行距与项目符号；标题/标签条目加 `b="1"`。切勿把多条拼进一个 `<a:t>`。

### 表格
表格是 `<a:tbl>`，逐个 `<a:tc>` 单元格内的 `<a:t>` 改文本。行数不符时整行增删 `<a:tr>`（注意同时调整 `<a:gridCol>` 列数一致）。

### 换学校品牌
- 校徽：替换 `ppt/media/image5.png`（branded 版式右上，587×193）与 `image2.png`（内容页左上小圆徽）。
- 校名文字：改封面与致谢中的“南通大学/学院/实验室”文本。
- 立方体背景 `image4.jpeg` 通常保留（中性学术风），如需换主题色改 theme1.xml 的 `293764`/`4A70AE`。

## 常见坑

- 删内容页时务必从 `<p:sldIdLst>` 删 `<p:sldId>`，再 `clean.py` 清理孤立媒体/rels。
- branded 页（封面/分隔/结束）本身几乎无图依赖（装饰在版式），复制最安全。
- 原模板含大量论文专属配图（框架图、截图、柱状图），新课题不适用——要么用用户的图替换，要么删图改文字，**不要留着原图**。
- ⚠️ **位图里的“烤死文字”是最隐蔽的残留**：slide5（流程图）、slide6（柱状图）、slide12/14/16（模型结构图）、slide19（系统截图）、slide11（贡献卡的结构缩略图）等页的 `<p:pic>` 都是位图，里面的旧模型名/旧指标/旧界面文字 **grep 查不到**，必须靠视觉质检看图发现。处理：删 `<p:pic>` + 删其 `.rels` 关系 + `clean.py`，原位放虚线占位框。文本残留则 grep 原答辩人名与原课题关键词确认清除。
- 中文文本框含引号时用 XML 实体（`&#x201C;` 等）。
